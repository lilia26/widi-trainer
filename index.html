Scioly WiDi Practice Kit

Master the Write It Do It Science Olympiad event through deliberate practice.

Introduction

The Science Olympiad Write It Do It Practice Kit is a web-based tool designed to help team members prepare for the Write It Do It event. In the event, one team member (the "Writer") is given a structure made from construction materials. The writer proceeds to write instructions on how to build the structure. The other team member (the "Doer") then uses the instructions and unassembled materials to try to recreate the structure as accurately as possible. The event depends on the team members to establish common vocabulary to be able to concisely and accurate describe various items under time constraints. Each kit that is included as part of this tool simulates specific aspects of the event experience to enable deliberate practice. The kits are an effective and efficient way for teams to practice and improve their Write It Do It skills.

Step 1: The tool generates 3 images, each containing 4 randomly positioned paperclips. The Doer must not be allowed to see the initial set of images. The Writer is required to write a description of each image.
Step 2: The tool generates 30 images, each containing 4 paperclips. The Doer must use the write-up created in Step 2 to locate and select the 3 original images.
Step 3: The tool then scores the answers and displays the results which can be used by the Writer and Doer to perform a retrospective.

Paperclip Kit - Write It

In this step, The Writer is presented with 3 random images of paperclips. The Doer should NOT see these images. The Writer may use the "Regenerate" button to recreate the images, if necessary. The Writer is expected to write a detailed description of each image on a separate piece of paper. This step helps the Writer practice their ability to clearly and accurately describe the paperclips, which is an important skill in the Write It Do It event.

Paperclip Kit - Do It

In this step, the Doer is presented with 30 images of paperclips, which will include the 3 original images from Step 1 as well as 27 additional images. The Doer is expected to use the descriptions provided by the Writer to locate and select the 3 original images among the 30 images presented. This step simulates the real-world scenario of the Write It Do It event, where the Doer will be required to use the descriptions provided by the writer to locate and construct the structures that use paperclips.

Paperclip Kit - Check It

In this step, the tool compares the answers provided by the Doer and matches them up to the original images described by the Writer. This information can be utilized to identify areas where mistakes were made and opportunities to improve. This step will help the team members to understand their strengths and weaknesses, and to identify areas to focus on in future practice sessions.
